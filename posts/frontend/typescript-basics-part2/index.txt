- **Part 1**: 기본 타입, 객체, 함수, Union/Intersection

## Type Narrowing (타입 좁히기)

TypeScript가 타입을 더 구체적으로 추론하도록 돕는 기법입니다. Union 타입을 다룰 때 필수적입니다.

### typeof 타입 가드

가장 기본적인 타입 좁히기 방법입니다. JavaScript의 `typeof` 연산자를 사용합니다.

```typescript
function processValue(value: string | number) {
  // 이 시점에서 value는 string | number

  if (typeof value === 'string') {
    // 이 블록 안에서 value는 string 타입
    return value.toUpperCase();
    // value.toFixed(2); //  에러- string에는 toFixed가 없음
  }

  // 이 블록 안에서 value는 number 타입 (else 블록)
  return value.toFixed(2);
  // value.toUpperCase(); // 에러- number에는 toUpperCase가 없음
}

// 더 복잡한 예시
function formatInput(input: string | number | boolean): string {
  if (typeof input === 'string') {
    return input.trim();
  }

  if (typeof input === 'number') {
    return input.toString();
  }

  // boolean만 남음
  return input ? 'Yes' : 'No';
}

console.log(formatInput('  hello  ')); // "hello"
console.log(formatInput(42)); // "42"
console.log(formatInput(true)); // "Yes"
```

주의- `typeof null === 'object'` (JS의 오래된 버그)

```typescript
function processValue(value: string | null) {
  if (typeof value === 'string') {
    return value.toUpperCase();
  }

  // 나쁜 예
  if (typeof value === 'object') {
    // null도 'object'이므로 여기 들어옴
    return 'null';
  }

  // 좋은 예
  if (value === null) {
    return 'null';
  }

  return value.toUpperCase();
}
```

### instanceof 타입 가드

클래스 인스턴스를 확인할 때 사용합니다.

```typescript
  bark() {
    console.log('Woof!');

class Cat {
  meow() {
    console.log('Meow!');
  }
}

function makeSound(animal: Dog | Cat) {
  if (animal instanceof Dog) {
    animal.bark(); // Dog 타입
  } else {
    animal.meow(); // Cat 타입
  }
}

const myDog = new Dog();
const myCat = new Cat();

makeSound(myCat); // "Meow!"
```

### in 연산자 타입 가드

객체에서 사용 가능한 타입 가드입니다.

```typescript
type Fish = {
  swim: () => void;
};

type Bird = {
  fly: () => void;
};

function move(animal: Fish | Bird) {
  if ('swim' in animal) {
    // Fish 타입
    animal.swim();
  } else {
    // Bird 타입
    animal.fly();
  }
}

const fish: Fish = {
  swim: () => console.log('Swimming...')
};

const bird: Bird = {
  fly: () => console.log('Flying...')
};

move(fish); // "Swimming..."
move(bird); // "Flying..."
```

실무에서는 옵셔널 속성 안전하게 접근

```typescript
interface User {
  name: string;
  email: string;
  address?: {
    city: string;
    country: string;
    zipCode?: string;
  };
}

function getUserLocation(user: User): string {
  // 나쁜 예: 옵셔널 체이닝만 사용
  return user.address?.city ?? 'Unknown';

  // 좋은 예: in 연산자로 명확하게 체크 - 더블 체크의 느낌
  if ('address' in user && user.address) {
    const { city, country } = user.address;

    if ('zipCode' in user.address && user.address.zipCode) {
      return `${city}, ${country} ${user.address.zipCode}`;
    }

    return `${city}, ${country}`;
  }

  return 'Location unknown';
}
```

### 사용자 정의 타입 가드

복잡한 타입 체크 로직을 함수로 캡슐화합니다. `value is Type` 형태의 반환 타입을 사용합니다.

```typescript
// 기본 타입 가드
function isString(value: unknown): value is string {
  return typeof value === 'string';
}

function isNumber(value: unknown): value is number {
  return typeof value === 'number';
}

// 사용
function processValue(value: unknown) {
  if (isString(value)) {
    console.log(value.toUpperCase()); // string
  } else if (isNumber(value)) {
    console.log(value.toFixed(2)); // number
  }
}
```

```typescript
interface User {
  id: number;
  name: string;
  email: string;
}

function isUser(obj: unknown): obj is User {
  return (
    typeof obj === 'object' &&
    obj !== null &&
    'id' in obj &&
    typeof (obj as User).id === 'number' &&
    'name' in obj &&
    typeof (obj as User).name === 'string' &&
    'email' in obj &&
    typeof (obj as User).email === 'string'
  );
}

async function fetchUser(id: number): Promise<User> {
  const response = await fetch(`/api/users/${id}`);
  const data: unknown = await response.json();

  if (isUser(data)) {
    return data; // User 타입 보장
  }

  throw new Error('Invalid user data');
}
```

```typescript
  return (
    Array.isArray(value) &&
    value.every(item => typeof item === 'string')
```

**제네릭 타입 가드**: 재사용 가능한 타입 가드(중요)

```typescript
// 배열의 모든 요소가 특정 타입인지 체크
function isArrayOf<T>(
  arr: unknown,
  check: (item: unknown) => item is T
): arr is T[] {
  return Array.isArray(arr) && arr.every(check);
}

// 사용
const data: unknown = ['a', 'b', 'c'];

if (isArrayOf(data, isString)) {
  // string[] 타입
  data.forEach(str => console.log(str.toUpperCase()));
}

// 숫자 배열 체크
const numbers: unknown = [1, 2, 3];

if (isArrayOf(numbers, isNumber)) {
  // number[] 타입
  const sum = numbers.reduce((a, b) => a + b, 0);
  console.log(sum);
}
```

실무 활용에서는 

```typescript
interface ApiResponse<T> {
  success: boolean;
  data?: T;
  error?: string;
}

function isApiResponse<T>(
  obj: unknown,
  dataCheck: (data: unknown) => data is T
): obj is ApiResponse<T> {
  if (typeof obj !== 'object' || obj === null) {
    return false;
  }

  const response = obj as ApiResponse<T>;

  // success는 필수
  if (typeof response.success !== 'boolean') {
    return false;
  }

  // data가 있으면 타입 체크
  if ('data' in response && response.data !== undefined) {
    if (!dataCheck(response.data)) {
      return false;
    }
  }

  // error가 있으면 string이어야 함
  if ('error' in response && typeof response.error !== 'string') {
    return false;
  }

  return true;
}

// 사용
async function fetchUser(id: number): Promise<User> {
  const response = await fetch(`/api/users/${id}`);
  const json: unknown = await response.json();

  if (isApiResponse(json, isUser)) {
    if (json.success && json.data) {
      return json.data; // User 타입
    }
    throw new Error(json.error || 'Unknown error');
  }

  throw new Error('Invalid API response');
}
```

## 제네릭 (Generics)

위에서 잠깐 나온 제네릭은 타입을 매개변수화하여 재사용 가능한 컴포넌트를 만드는 도구입니다.

### 제네릭이 필요한 이유

제네릭 없이는 같은 로직을 여러 타입에 대해 중복 작성하거나, `any`를 사용해야 합니다.

```typescript
// 타입별로 함수 중복
function getFirstString(arr: string[]): string | undefined {
  return arr[0];
}

function getFirstNumber(arr: number[]): number | undefined {
  return arr[0];
}

function getFirstBoolean(arr: boolean[]): boolean | undefined {
  return arr[0];
}

// any 사용 - 타입 안정성 상실
function getFirst(arr: any[]): any {
  return arr[0];
}

const first = getFirst([1, 2, 3]); // any 타입
 // 런타임 에러

// 제네릭 사용
function getFirst<T>(arr: T[]): T | undefined {
  return arr[0];
}

const firstNum = getFirst([1, 2, 3]); // number | undefined
const firstStr = getFirst(['a', 'b', 'c']); // string | undefined
const firstBool = getFirst([true, false]); // boolean | undefined
```

### 기본 제네릭

```typescript
// 기본 제네릭 함수
function identity<T>(value: T): T {
  return value;
}

// 명시적 타입 지정
const num = identity<number>(42); 
const str = identity<string>('hello');

// 타입 추론 
const auto = identity(true); // boolean - 자동 추론

// 배열 반전 함수
function reverse<T>(arr: T[]): T[] {
  return arr.reverse();
}

reverse([1, 2, 3]); // [3, 2, 1] - 숫자 배열
reverse(['a', 'b', 'c']); // ['c', 'b', 'a'] - 문자열 배열 

// 여러 제네릭 파라미터
function pair<T, U>(first: T, second: U): [T, U] {
  return [first, second];
}

const p1 = pair(1, 'hello'); // [number, string]
const p2 = pair('id', { name: 'Alice' }); // [string, { name: string }]

// 튜플 첫 번째 요소 반환
function getFirstOfPair<T, U>(pair: [T, U]): T {
  return pair[0];
}

getFirstOfPair([1, 'hello']); 
getFirstOfPair(['id', 42]); 
```

```typescript
// 제네릭 인터페이스
interface Box<T> {
  value: T;

const numBox: Box<number> = { value: 42 };
const strBox: Box<string> = { value: 'hello' };
const userBox: Box<User> = { value: { id: 1, name: 'Alice', email: 'a@b.com' } };

interface Pair<K, V> {
  key: K;
  value: V;
}

const idName: Pair<number, string> = {
  key: 1,
  value: 'Alice'
};

// 제네릭 타입 별칭 - 기본값 제공
type Result<T, E = Error> =
  | { success: true; value: T }
  | { success: false; error: E };

// E를 생략 시 Error가 기본값
const success: Result<User> = {
  success: true,
  value: { id: 1, name: 'Alice', email: 'a@b.com' }
};

// E를 명시적으로 지정
const failure: Result<User, string> = {
  success: false,
  error: 'User not found'
};
```

```typescript
class Stack<T> {

  push(item: T): void {
    this.items.push(item);
  }

  pop(): T | undefined {
    return this.items.pop();
  }

  peek(): T | undefined {
    return this.items[this.items.length - 1];
  }

  get size(): number {
    return this.items.length;
  }

  isEmpty(): boolean {
    return this.items.length === 0;
  }

  clear(): void {
    this.items = [];
  }
}

// 사용
const numberStack = new Stack<number>();
numberStack.push(1);
numberStack.push(2);
numberStack.push(3);

console.log(numberStack.pop()); // 3
console.log(numberStack.peek()); // 2
console.log(numberStack.size); // 2

const stringStack = new Stack<string>();
stringStack.push('hello');
stringStack.push('world');

console.log(stringStack.pop()); // 'world'
```

```typescript
// extends로 제약
interface HasId {
  id: number;
}

function findById<T extends HasId>(items: T[], id: number): T | undefined {
  return items.find(item => item.id === id);
}

// HasId를 만족하는 타입들
const users = [
  { id: 1, name: 'Alice', email: 'alice@example.com' },
  { id: 2, name: 'Bob', email: 'bob@example.com' }

// HasId를 만족하지 않는 타입
```

```typescript
  return obj[key];
}

  age: 25,

};

function pick<T, K extends keyof T>(obj: T, keys: K[]): Pick<T, K> {
  const result = {} as Pick<T, K>;
  for (const key of keys) {
    result[key] = obj[key];
  }
  return result;
}

const userSubset = pick(user, ['name', 'email']);
// { name: string; email: string }
```

```typescript
async function fetchWithOptions<T>(
  url: string,
  options?: FetchOptions
): Promise<T> {
  let fullUrl = url;

  // 쿼리 파라미터 추가

  if (!response.ok) {
    throw new Error(`HTTP error! status: ${response.status}`);
  }

  return response.json();
}
```

```typescript
class Cache<K, V> {
  private store = new Map<K, V>();
  private maxSize: number;

  constructor(maxSize: number = 100) {
    this.maxSize = maxSize;
  }

  set(key: K, value: V): void {
    // 크기 제한
    if (this.store.size >= this.maxSize && !this.store.has(key)) {
      // LRU: 가장 오래된 항목 제거
      const firstKey = this.store.keys().next().value;
      this.store.delete(firstKey);
    }

    this.store.set(key, value);
  }

  get(key: K): V | undefined {
    return this.store.get(key);
  }

  has(key: K): boolean {
    return this.store.has(key);
  }

  delete(key: K): boolean {
    return this.store.delete(key);
  }

  clear(): void {
    this.store.clear();
  }

  get size(): number {
    return this.store.size;
  }
}

// 사용
const userCache = new Cache<number, User>(50);

userCache.set(1, { id: 1, name: 'Alice', email: 'alice@example.com' });
userCache.set(2, { id: 2, name: 'Bob', email: 'bob@example.com' });

const user = userCache.get(1); // User | undefined

if (user) {
  console.log(user.name); // 'Alice'
}
```

```typescript
// 재시도 로직
async function retry<T>(
  fn: () => Promise<T>,
  maxAttempts: number = 3,
  delay: number = 1000
): Promise<T> {
  let lastError: unknown;

  for (let attempt = 1; attempt <= maxAttempts; attempt++) {
    try {
      return await fn();
    } catch (error) {
      lastError = error;

      if (attempt === maxAttempts) {
        throw error;
      }

      console.log(`Attempt ${attempt} failed, retrying in ${delay}ms...`);
      await new Promise(resolve => setTimeout(resolve, delay));
    }
  }

  throw lastError;
}

// 사용
const data = await retry(
  () => fetchData<User>('/api/user/1'),
  3,
  2000
);

// 타임아웃 추가
async function withTimeout<T>(
  promise: Promise<T>,
  timeoutMs: number
): Promise<T> {
  const timeout = new Promise<never>((_, reject) => {
    setTimeout(() => {
      reject(new Error(`Timeout after ${timeoutMs}ms`));
    }, timeoutMs);
  });

  return Promise.race([promise, timeout]);
}

// 사용
const user = await withTimeout(
  fetchData<User>('/api/user/1'),
  5000 
);

// 조합
const userWithRetry = await retry(
  () => withTimeout(fetchData<User>('/api/user/1'), 5000),
  3,
  2000
);
```

```typescript
type EventHandler<T> = (data: T) => void;

class EventEmitter<T extends Record<string, any>> {
  private listeners = new Map<keyof T, Set<EventHandler<any>>>();

  on<K extends keyof T>(event: K, handler: EventHandler<T[K]>): void {
    if (!this.listeners.has(event)) {
      this.listeners.set(event, new Set());
    }
    this.listeners.get(event)!.add(handler);
  }

  off<K extends keyof T>(event: K, handler: EventHandler<T[K]>): void {
    const handlers = this.listeners.get(event);
    if (handlers) {
      handlers.delete(handler);
    }
  }

  emit<K extends keyof T>(event: K, data: T[K]): void {
    const handlers = this.listeners.get(event);
    if (handlers) {
      handlers.forEach(handler => handler(data));
    }
  }

  once<K extends keyof T>(event: K, handler: EventHandler<T[K]>): void {
    const onceHandler: EventHandler<T[K]> = (data) => {
      handler(data);
      this.off(event, onceHandler);
    };
    this.on(event, onceHandler);
  }
}

// 사용
interface AppEvents {
  userLogin: { userId: number; timestamp: Date };
  userLogout: { userId: number };
  dataUpdate: { type: string; data: unknown };
}

const events = new EventEmitter<AppEvents>();

events.on('userLogin', (data) => {
  console.log(`User ${data.userId} logged in at ${data.timestamp}`);
});

events.emit('userLogin', {
  userId: 123,
  timestamp: new Date()
});

// 잘못된 데이터 전달 시 에러
events.emit('userLogin', { userId: '123' }); // 에러
```
